## Descrição

Esta API permite controlar mesas, produtos e comandas de um restaurante, 
permitindo adicionar itens e realizar o fechamento da comanda.

## Tecnologias utilizadas

- ASP.NET Core
- Entity Framework Core
- SQLite

## Funcionalidades

- Cadastro de mesas
- Cadastro de produtos
- Criação de comandas
- Adição de itens nas comandas
- Fechamento de comandas
